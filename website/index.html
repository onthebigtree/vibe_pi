Built to be glanced at.

Real budget, not guesswork

Reads your actual session token usage and renders the 5-hour and 7-day windows as two nested rings. The big number is what's left.

Warns only when it matters

Calm by default — cool white, never a wall of color. It speaks up with amber and red only as you near a limit, so a warning actually reads as one.

USB or Wi-Fi

Plug it in over USB for a zero-config wired link, or run it wireless on your network. It falls back automatically and shows the active mode.

Local-first & private

A small agent runs on your machine and talks to the device directly. No cloud account, no telemetry — your usage stays on your network.

Multi-tool aware

Running Claude and Cursor at once? Tap to cycle tools. One device, your whole AI toolbelt.

Updates over the air

Signed firmware with automatic rollback: a bad update reverts itself, so the device on your desk just keeps working.

Three steps to live.

  1. 1

    Place it

    Set the round display on your desk and power it over USB-C.

  2. 2

    Run the agent

    Install the menu bar app (macOS) or run vibe-pi-host. It finds your AI tools automatically.

  3. 3

    Pair & go

    Confirm the 6-digit code once. From then on it just shows up.
