Questions, answered.

Is my usage data private?

Yes. Vibe Pi is local-first. A small agent runs on your own machine, reads your tools' usage locally, and talks to the device directly over USB or your LAN. There's no cloud account and no telemetry — nothing leaves your network.

Which tools does it track?

Claude Code and Codex report real usage today — the 5-hour and 7-day window percentages. Cursor, Windsurf and Gemini CLI are detected as running, with fuller usage numbers coming in a future update.

USB or Wi-Fi — which should I use?

USB-C serial is the primary link: plug it in for a zero-config wired connection. Wi-Fi (WebSocket) is the automatic fallback if USB isn't available. The device shows which mode is active.

How do I pair it?

The device shows a 6-digit code. Confirm it once in the dashboard and you're done — after that it reconnects on its own.

How do updates work?

Over the air, signed, with automatic rollback. If an update fails to boot cleanly, the device reverts to the previous firmware on its own — the monitor on your desk just keeps working.

What do I need to run it?

The host agent runs on macOS, Linux or Windows. On macOS there's a menu bar app; everywhere there's the vibe-pi-host CLI. See the Download page to get started.
